Gift Wrapping

An implementation of the Gift Wrapping algorithm, which draws a convex hull for a given set of points.
Fullscreen

This program visualizes the "gift wrapping algorithm" which will generate a convex hull for a given set of points.
A convex hull is the smallest area with a convex curve that can contain a given set of points.
Convex means that it will never dip inward, it will expand if need be, but never contract.
